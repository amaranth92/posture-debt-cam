# 디스크 수술비 캠

웹캠으로 사용자의 자세를 추정해서 거북목/구부정 지수가 올라갈수록 화면을 뿌옇게 만들고, 우측 하단에 가상의 목/허리 디스크 수술비 청구서를 띄우는 바이럴 데모입니다.

> 의료 진단용이 아닙니다. MediaPipe 2D pose landmark 기반의 교육/엔터테인먼트용 휴리스틱 데모입니다.

## 기능

- 웹캠 실시간 자세 감지
- 거북목 지수 / 구부정 지수 계산
- 자세가 나빠질수록 화면 blur/fog 증가
- 우측 하단 가상 수술비 청구서
- 단계 상승 시 돈통/동전 느낌 WebAudio 효과음
- 브라우저만으로 실행 가능한 정적 웹앱

## 실행

```bash
python -m http.server 8765
```

브라우저에서 열기:

```text
http://127.0.0.1:8765
```

웹캠 권한을 허용해야 동작합니다.

## 파일

```text
index.html
styles.css
app.js
```

## 배포

GitHub Pages, Netlify, Vercel 같은 정적 호스팅에 그대로 올릴 수 있습니다. 웹캠 API는 HTTPS 또는 localhost에서 동작합니다.
